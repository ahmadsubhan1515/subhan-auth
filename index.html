Use these endpoints to integrate SUBHAN AUTH with your applications:

C# Integration Example:

string apiUrl = "https://your-domain.com/api/validate";
string key = "USER_KEY_HERE";
string hwid = Environment.MachineName; // or custom HWID

var client = new HttpClient();
var response = await client.PostAsync(apiUrl, new StringContent(
  JsonConvert.SerializeObject(new { key, hwid }),
  Encoding.UTF8, "application/json"));

var result = await response.Content.ReadAsStringAsync();
var authResult = JsonConvert.DeserializeObject<AuthResponse>(result);

Discord Bot Command:

/newkey duration:30 type:days hwid:true

API Endpoints:

POST /api/validate - Validate a key
POST /api/create - Create new key (admin only)
POST /api/revoke - Revoke a key (admin only)
GET /api/stats - Get key statistics
